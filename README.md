## 🖼️ Как добавить фотографии

### Структура изображений

Проект организован для 4 категорий дресс-кода, каждая с каруселью (слайдером):

- **Мужчины** — ~5-10 фото образцов для мужчин
- **Женщины** — ~5-10 фото образцов для женщин  
- **Пары** — ~5-10 фото пар в праздничных нарядах
- **Дети** — ~5-10 фото детей в нарядах

### Алгоритм добавления фото

1. **Создайте папку для фото** (опционально):
   ```
   assets/images/
   ├── men/
   ├── women/
   ├── couples/
   └── kids/
   ```

2. **Оптимизируйте фотографии**:
   - Размер: ~480×640px (пропорция 3:4)
   - Формат: JPG/WebP для быстрой загрузки
   - Качество: ~85% для баланса между размером и качеством

3. **Замените плейсхолдеры в HTML**:

**Для мужчин** (найти строки с `Men+1`, `Men+2` и т.д.):
```html
<!-- БЫЛО: -->
<img src="https://placehold.co/480x640/2F4156/F5EFEB?text=Men+1" alt="Образец для мужчин 1" class="carousel-slide ...">

<!-- СТАНЕТ: -->
<img src="assets/images/men/look-1.jpg" alt="Образец для мужчин 1" class="carousel-slide ...">
```

**Для женщин** (поиск `Women+1`, `Women+2` и т.д.):
```html
<img src="assets/images/women/look-1.jpg" alt="Образец для женщин 1" class="carousel-slide ...">
```

**Для пар** (поиск `Couple+1`, `Couple+2` и т.д.):
```html
<img src="assets/images/couples/look-1.jpg" alt="Образец для пар 1" class="carousel-slide ...">
```

**Для детей** (поиск `Kids+1`, `Kids+2` и т.д.):
```html
<img src="assets/images/kids/look-1.jpg" alt="Образец для детей 1" class="carousel-slide ...">
```

## 🎵 Как добавить фоновую музыку

1. **Подготовьте аудиофайл**:
   - Формат: MP3 (поддерживается всеми браузерами)
   - Длительность: 2-4 минуты (для зацикливания)
   - Качество: 128-192 kbps достаточно для фона

2. **Разместите файл**:
   ```
   assets/music/twins-lullaby.mp3
   ```

3. **Путь в HTML уже готов**:
   - В строке `<audio id="bgMusic" loop>` уже указан путь `assets/music/twins-lullaby.mp3`
   - Просто положите файл туда — плеер заработает автоматически

### Как работает музыкальный плеер

- 🔘 **Кнопка** в правом нижнем углу (синий круг)
- **Автозапуск**: музыка попытается включиться сразу, если браузер позволит
- **Фолбэк**: если браузер блокирует автоплей, музыка включится по первому клику/тапу гостя
- **Иконка**: вращающийся диск → при воспроизведении переключается на анимированный эквалайзер
- **Громкость**: установлена на 50% для комфортного прослушивания

## 🎨 Цветовая палитра

Сайт использует благородную палитру:

```css
Navy (Темно-синий)      → #2F4156  (заголовки, кнопки, акценты)
Teal (Приглушённый)     → #567C8D  (вспомогательные элементы)
Sky (Небесно-голубой)   → #C8D9E6  (мягкие фоны)
Beige (Благородный)     → #F5EFEB  (основной фон страницы)
White                   → #FFFFFF  (карточки, контраст)
```

## 📱 Адаптивность

- **Мобильные** (< 640px): 1 слайд в карусели, полная ширина
- **Планшеты** (640px-1024px): 2-3 слайда видимо
- **Десктопы** (> 1024px): 3 слайда видимо одновременно

## 🚀 Развёртывание

1. Убедитесь, что все фотографии добавлены в `assets/images/`
2. Убедитесь, что музыка находится в `assets/music/twins-lullaby.mp3`
3. Откройте `index.html` в браузере или загрузите на хостинг

## 💡 Советы

- **Фото**: используйте портретную ориентацию (3:4) для лучшего вида на мобильных
- **Musik**: выбирайте нежные, спокойные треки (колыбельные, музыкальные шкатулки)
- **Сессии**: фоновая музыка автоматически пауза-resume при закрытии вкладки
- **Тёмный режим**: сайт адаптирован под светлый режим (Beige фон), но хорошо выглядит везде
